# shell-scripts-toolkit
A collection of production-ready Bash scripts for Linux, Docker, k8s, AWS, and Devops Automation

A curated collection of production-ready Bash scripts tried designing to automate common Devops and system administration tasks.

This repository focuses on practical automation used in Linux environments, Kubernetes clusters, Docker hosts, AWS infra and day-to-day operational support.

Objectives:
Automate repetitive operational tasks
Improve system visibility and health checks
Simplify Kubernetes administration
Demonstrate production-ready Bash scripting practices
